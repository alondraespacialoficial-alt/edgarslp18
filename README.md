# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/9b963d54-9586-4ca7-b030-839da22479de

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Copia [.env.example](.env.example) a `.env` y completa tus propias credenciales (Gemini API key, ADMIN_PASSWORD y, opcionalmente, las de Supabase):
   `cp .env.example .env`
3. Run the app:
   `npm run dev`
